使用Pycharm连接远程服务器，并在本地进行调试

连接远程服务器

本机环境:
Win10 + Pycharm Professional 2019.2

首先确保你的Pycharm是Professional版本，而不是Community版本，不然按照网上的教程你会发现你找不到相应的功能。

首先在菜单栏找到Tools,下拉栏中依次选择Deployment -> Configuration, 点击左上角的+， 选择SFTP,然后填写IP地址，端口号，用户名和密码，填好之后点击Test Connection, 应该就能显示连接成功了。

接下来在Root Path中选择你要将本地代码上传到服务器的根目录，然后选择Mappings一栏， 填写Local path和Deployment path(相对于刚才设置的Root path)，之后的文件上传和下载都是在这两个路径之间进行，填好后点击OK就基本设置完成了。

上传和下载

可以自动同步也可以手动同步。想要自动同步的话，勾选Tools -> Deployment -> Automatic Upload,勾选之后每次保存修改都会触发自动上传。

手动同步的话，在你想要上传的文件夹处右键然后选择Deployment -> Upload to .../Download from ...即可上传或下载文件。

配置远程python解释器

从菜单栏选择File, 然后依次选择Project -> Project Interpreter，可以看到右边有个小齿轮，点击并依次选择Add... -> SSH Interpreter。 如果前面已经连接好了服务器，直接勾选Existing server configuration, 否则在这里填入服务器的IP地址和端口和用户名，点击next, 然后填入密码。 再点击next, 选择好python解释器， 以及相应的路径。这里还要注意如果是手动同步文件的话， 取消掉勾选框， 然后点击Finish就完成了，

本地调试

接下来点击Pycharm右上角的绿色小三角应该就可以使用远程的服务器来运行程序了， 同样也可以使用debug工具在本地调试。每次在本地修改代码后， 运行之前记得使用upload to ...把代码同步到服务器。而运行程序生成的文件也可以使用Download from ...下载到本地，这样方便查看和修改。
